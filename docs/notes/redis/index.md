# Redis 总览

Redis 值得单独成一条线，因为它不是一个“顺手加上的缓存组件”，而是一组完整的工程能力：

- 缓存设计
- 数据结构选型
- 一致性控制
- 热点治理
- 分布式锁
- 计数、限流、排行榜

所以这一组不准备只写一篇“Redis 是什么”，而是沿着真实项目里最常遇到的问题往下拆。
