## 建议分类

- `项目模板`：Spring Boot 基础结构、依赖、配置和分层
- `接口设计`：Controller、DTO、返回结构、异常约定
- `服务实现`：Service、事务、缓存、消息、异步任务
- `排障手册`：日志、链路、数据库、性能、线上问题
