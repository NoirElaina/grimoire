# Claude 源码解析

这个分支记录对 `Claude Code` 泄露事件的高层拆解。

这里不复述泄露代码，也不讨论如何获取泄露内容，而是把它当成一个观察窗口：看看一款成熟 coding agent 产品，真正有价值的部分大多落在哪些层。

## 目录

- [01 Claude Code 泄露事件与架构启示](/notes/agents/claude-code-analysis/first-look)

## 这个分支想回答什么

- `Claude Code` 这次公开暴露出来的，究竟是不是“模型核心”？
- 一款成熟 Agent 产品，除了模型本身，还有哪些产品层和工程层能力？
- 如果我们自己做 Agent，真正值得学的东西是什么？

## 写法约定

- 只基于公开报道、公开页面和公开讨论做高层分析
- 不贴泄露代码片段
- 不把“看见实现细节”误读成“拿到了完整能力”
